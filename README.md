## üß© System Modules

The application is divided into **two main sections**:



### üìò 1. Menu Section

This section is used to manage book records in the system.

* **Insert New Books**

  * Enter book details such as **title** and **price**.

* **Update Book Details**

  * Modify existing book information.

* **Delete Books**

  * Remove book records from the system.

* **Clear**

  * Clear all selected input fields.

* **Search Books**

  * Search books by **book name**.

* **View All Books**

  * Display the complete list of available books.



### üßæ 2. Billing Section

This section is used to generate and manage customer bills.

* **Bill Date & Bill Number**

  * Automatically sets the **current date** and **bill number**.

* **Customer Name & Contact Number**

  * Enter customer name and phone number.

* **Quantity**

  * Enter the quantity required by the customer (e.g., 1, 2, 3‚Ä¶).
  * Press **Enter** to add the product to the billing table (cart).

* **Delete Selected Row**

  * Select a row and remove the item from the cart.

* **Discount**

  * Shopkeeper can manually apply a **percentage discount (%)**.

* **Save & Print Bill**

  * Saves the bill details and prints the customer invoice.
